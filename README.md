# 👟 MiroMiro System v1.0
A nostalgic, retro-inspired Y2K footwear storefront designed to look and feel like a vintage operating system. Built with Vanilla JS, CSS 'checkbox hacks', and LocalStorage.

## 💿 Features
* **Retro OS Interface:** Navigation through a "Start Menu" and draggable-style windows.
* **Smart Carousel:** A custom-built, auto-scrolling product showcase with manual navigation and hover-detection.
* **Persistent Basket:** A fully functional shopping cart system using `localStorage` to keep data across pages.
* **Interactive Modals:** Product details and shop views handled via high-performance CSS and JavaScript.
* **Responsive Design:** Optimized for both the "Desktop OS" feel and mobile "System" view.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure with custom data attributes.
* **CSS3:** Heavy use of Flexbox, Grid, and the "Checkbox Hack" for state management without heavy libraries.
* **Vanilla JavaScript:** Custom event listeners for scrolling, carousel logic, and basket calculations.
